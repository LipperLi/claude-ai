# Claude AI - Next.js App 🦞

一个基于 Next.js 15 的 Claude AI 展示应用，带有可爱的小龙虾动画！

## 🚀 技术栈

- **Next.js 15.1.6** - React 服务端渲染框架
- **React 19** - 最新版本的 React
- **TypeScript** - 类型安全
- **Tailwind CSS 3.4** - 实用优先的 CSS 框架

## ✨ 特性

- 🦞 弹跳的小龙虾图标动画
- ⏰ 实时时钟显示
- 🎨 现代渐变背景设计
- 📱 响应式布局

## 🛠️ 安装与运行

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 生产构建
pnpm build

# 启动生产服务器
pnpm start
```

应用将运行在 `http://localhost:3000`
